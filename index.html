Siri Shortcuts

My intern project for my summer at Vimeo was to create a Siri Shortcut for the iOS application using Swift. Vimeo is a completely ad-free video platform and they have multiple subscription tiers for users such as professional videographers to upload and own their content. The Vimeo Siri Shortcut feature that I created learns which privacy setting a user was uploading videos with most frequently in the Vimeo iOS application. Then the user will receive a notification to upload a video with their most used privacy setting. By clicking on this notification, the user is immediately brought to the Vimeo upload flow with the privacy setting already populated. I was also able to localize this feature by making it available in all the languages that Vimeo supports. This feature is currently in production in the Vimeo iOS application.

Math Team Web Application

This web application is a collection of helpful tools for both math team coaches and students to practice questions and get updates on upcoming activities. To create this application, I utilized user authentication so that students and coaches can log into the website and practice questions. Coaches can then view students' accuracy rates and question attempts from the admin view. This application was built using the Python framework, Django, as well as HTML, CSS, and Bootstrap.

Politigo

This web application allows users to search a database of political bills and politicians to find information on current and past political climates. The backend was created with Node.js, GraphQL and the ProPublic Congress API and the frontend was created with React.
